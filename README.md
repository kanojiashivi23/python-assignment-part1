# Student Grade Tracker

This project is a simple Python-based Student Grade Tracker. It works with raw student data, cleans it, analyzes marks, and generates useful outputs like grades and class reports.

## 📌 What this project does

### Task 1: Data Cleaning
- Cleaned student names by removing extra spaces and formatting properly
- Converted roll numbers from string to integer
- Converted marks from string to a list of integers
- Checked if names are valid (only alphabets)
- Printed student details in a formatted way

### Task 2: Marks Analysis
- Assigned grades (A+, A, B, C, F) based on marks
- Calculated total and average marks
- Found highest and lowest scoring subjects
- Added new subjects using a while loop
- Handled invalid inputs (like wrong marks)

### Task 3: Class Performance Summary
- Calculated average marks for each student
- Determined Pass/Fail status
- Printed a class report in table format
- Counted how many students passed and failed
- Found class topper
- Calculated overall class average

### Task 4: String Manipulation
- Cleaned an essay by removing extra spaces
- Converted text into title case
- Counted how many times "python" appears
- Replaced words with formatted text
- Split essay into sentences and printed them

## 🛠️ Concepts Used
- Loops (for, while)
- If-else conditions
- Lists and strings
- Basic functions like sum(), max(), min()
